# 0448 - Find All Numbers Disappeared in an Array

**Difficulty:** Easy  
**Status:** Planned  
**Last Seen:** none yet  

## Problem Statement

Given an array `nums` of `n` integers where each value is in the range `[1, n]`, return all the numbers in the range that are missing from the array.

This is a natural follow-up to 0442 because it uses the same in-place sign-marking trick, but the ask is inverted: missing values instead of duplicates.
