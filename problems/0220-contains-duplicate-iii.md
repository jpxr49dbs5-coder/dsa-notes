# 0220 - Contains Duplicate III

**Difficulty:** Hard  
**Status:** Planned  
**Last Seen:** none yet  

## Problem Statement

Given an integer array `nums` and two integers `k` and `t`, return `true` if there are two distinct indices `i` and `j` such that `abs(nums[i] - nums[j]) <= t` and `abs(i - j) <= k`.

This combines index and value constraints, making it the hardest in the Contains Duplicate family.
