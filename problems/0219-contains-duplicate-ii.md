# 0219 - Contains Duplicate II

**Difficulty:** Easy  
**Status:** Planned  
**Last Seen:** none yet  

## Problem Statement

Given an integer array `nums` and an integer `k`, return `true` if there are two distinct indices `i` and `j` such that `nums[i] == nums[j]` and `abs(i - j) <= k`.

This adds an index constraint to 0217, requiring a sliding window approach.
