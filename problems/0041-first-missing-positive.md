# 0041 - First Missing Positive

**Difficulty:** Hard  
**Status:** Planned

## Problem Statement

Given an unsorted integer array `nums`, return the smallest missing positive integer.

This is the next escalation after 0442/0448: it still lives in the same index-as-hash family, but the lack of a strict `[1, n]` placement guarantee makes the problem harder.
